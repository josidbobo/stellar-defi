## SETUP INSTRUCTIONS

1. Clone the repo
### `git clone https://github.com/josidbobo/stellar-defi`

2. Install dependencies
### `npm install`

3. Start the client
### `npm start`

4. Generate build folder to deploy
### `npm run build`

Builds the app for production to the `build` folder.\
